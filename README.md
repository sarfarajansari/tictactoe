## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sarfarajansari/tictactoe.git
   cd tictactoe
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
   The game will be available at `http://localhost:3000`.
